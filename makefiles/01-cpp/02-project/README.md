## Usage Executable Project Building Makefile
* Library name should start with "**prj**" prefix.
* You can find an example executable project folder hierarchy.
```C++
    * prjSortAndDisplay     /* (folder) */
        * 01-inc            /* (folder) */
        * 02-src            /* (folder) */
        * 03-obj            /* (folder) */
        * makefile
```
